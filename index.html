FAQ
Questions answered
Is FlowPDF really free? +
Yes, completely. There are no paid plans, no premium features, no watermarks, and no credit card required. All tools are free to use as many times as you want.
Are my files safe? +
Absolutely. Your files never leave your device. All processing is done locally in your browser using JavaScript (pdf-lib and PDF.js). We have no servers that receive your files.
How does the Smart Color Erase work? +
When you select an area to erase, FlowPDF samples the pixel color at the center of your selection. It then fills the erased area with that exact color — making the deletion invisible rather than leaving a white box.
Is there a file size or page limit? +
No limits. You can merge as many PDFs as you want and edit documents of any size. The only constraint is your device's available memory, which is rarely an issue for normal PDFs.
Does it work on mobile? +
Yes. Both the PDF Merger and PDF Editor are fully touch-friendly and work on iPhones, Android phones, and tablets. The editor supports touch drawing and touch selection.
Can I edit password-protected PDFs? +
FlowPDF attempts to open encrypted PDFs with the ignoreEncryption flag, which works for some lightly protected files. Strongly password-protected PDFs will require the password to be removed first by the document owner.
